### Logon event type
----
|Type|Title|Description|
|:----:|:----|:----|
|2|互動|使用者已登入這部電腦|
|3|網路|使用者或電腦從網路登入這部電腦|
|4|批次|批次登入類型會使用批次的伺服器，代表其直接介入使用者執行處理程序可能地方|
|5|服務|服務已啟動的服務控制管理員|
|7|解除鎖定|輸入密碼登入使工作站解除鎖定|
|10|RemoteInteractive|使用者使用[終端機服務]或[遠端桌面]從遠端登入這部電腦|
